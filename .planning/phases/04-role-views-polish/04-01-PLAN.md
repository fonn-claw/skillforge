**1. CREATE app/api/mentor/reviews/route.ts** — GET pending reviews:
- Verify mentor/admin role
- Query challengeSubmissions WHERE status = 'pending'
- JOIN with users (for learner name), challenges (for title, nodeId, masteryLevel, type)
- Return: [{ id, learnerName, learnerEmail, challengeTitle, challengeType, masteryLevel, response, submittedAt }]
- Order by submittedAt ASC (oldest first)

**2. CREATE app/api/mentor/reviews/[id]/route.ts** — POST review action:
- Verify mentor/admin role
- Parse body: { action: 'approve' | 'reject', feedback?: string }
- Update challengeSubmission: set status to action, mentorId = userId, feedback, reviewedAt = now
- If approved and type is project_submission:
  - Award XP same as quiz: xpPerLevel based on challenge masteryLevel
  - Full XP for approved projects (score = 100)
  - Update userNodeMastery (same logic as quiz submit)
  - Check for level-up
  - Return { updated: true, xpAwarded, leveledUp }
- If rejected:
  - Return { updated: true, status: 'reject' }

**3. CREATE components/mentor/ReviewPanel.tsx** — Review interface:

  - Simplest: slide-in from left side, opposite of NodeDetailPanel
