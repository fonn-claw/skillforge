- Define `nodeTypes` and `edgeTypes` OUTSIDE the component as module-level constants (CRITICAL — defining inside causes unmount/remount on every render):
  ```
  const nodeTypes = { hexagon: HexagonNode } as const;
  const edgeTypes = { prerequisite: PrerequisiteEdge } as const;
  ```
- Define mastery level order array: `['locked', 'novice', 'apprentice', 'journeyman', 'expert', 'master']`
- Define `getEdgeStatus(edge, masteryMap)` helper:
  - If source node has no mastery or mastery is 'locked' → 'inactive'
  - If source node mastery level index >= required mastery level index → 'active'
  - If target node also has mastery (not null and not 'locked') → 'completed'
  - Default: 'inactive'
- Define `toFlowNodes(apiNodes)` function: map API nodes to React Flow nodes with `type: 'hexagon'`, `position: { x: positionX, y: positionY }`, and data containing name, mastery (defaulting to 'locked' if null), iconKey, branchName, description, xpCurrent, xpRequired
- Define `toFlowEdges(apiEdges, masteryMap)` function: map API edges to React Flow edges with `type: 'prerequisite'`, source/target from sourceNodeId/targetNodeId, and data with computed status
- Component state: `useNodesState`, `useEdgesState`, `selectedNodeId` (string|null), `loading` (boolean)
- useEffect on mount: fetch both `/api/tree/nodes` and `/api/tree/edges` with `Promise.all`, transform to flow format, set nodes and edges. Set loading=false when done.
- `onNodeClick` callback: set `selectedNodeId` to node.id (will be used by NodeDetailPanel in Plan 02)
- `onPaneClick` callback: clear `selectedNodeId`
- Export `selectedNodeId` and `setSelectedNodeId` via a React context or pass as props (use a simple context: `export const TreeSelectionContext = createContext<{selectedNodeId: string|null; setSelectedNodeId: (id: string|null) => void}>({selectedNodeId: null, setSelectedNodeId: () => {}})`)
- Render structure:
  ```
  <TreeSelectionContext.Provider value={{ selectedNodeId, setSelectedNodeId }}>
    <div className="relative w-full h-full">
      <ReactFlow
        nodes={nodes}
        edges={edges}
        onNodesChange={onNodesChange}
        onEdgesChange={onEdgesChange}
        nodeTypes={nodeTypes}
        edgeTypes={edgeTypes}
        onNodeClick={onNodeClick}
        onPaneClick={onPaneClick}
        minZoom={0.3}
        maxZoom={2.0}
        fitView
        fitViewOptions={{ padding: 0.2 }}
        nodesDraggable={false}
        nodesConnectable={false}

        <Controls

          showInteractive={false}
        />
      </ReactFlow>
    </div>
  </TreeSelectionContext.Provider>
  ```
