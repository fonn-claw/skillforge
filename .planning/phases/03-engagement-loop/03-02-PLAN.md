**1. app/api/challenges/[id]/route.ts** — GET challenge with content:
- Import db, challenges from schema, eq from drizzle-orm
- Get challengeId from params.id
- Query challenge by id, return ALL fields including content
- Return { id, title, description, type, masteryLevel, content, nodeId }
- 404 if not found

**2. app/api/challenges/[id]/submit/route.ts** — POST challenge submission:
- Import db, challenges, challengeSubmissions, userNodeMastery from schema
- Get userId from headers (x-user-id)
- Get challengeId from params.id
- Parse body based on challenge type:
  - Quiz: { answers: number[] } — array of selected option indices
  - Project: { description: string, url?: string }
- Load the challenge to get its type, content, nodeId, masteryLevel
- If quiz:
  - Validate answers against content.questions[].correctIndex
  - Score = number of correct / total questions
  - Calculate XP: xpPerLevel = { novice: 25, apprentice: 35, journeyman: 50, expert: 75, master: 100 }
  - earnedXP = Math.round(score * xpPerLevel[challenge.masteryLevel])
  - Create challengeSubmission with status 'completed', score (0-100), response: { answers, score }
  - Update userNodeMastery: add earnedXP to xpCurrent
  - Check if xpCurrent >= xpRequired: if so, advance currentLevel, reset xpCurrent=0, set new xpRequired
  - New xpRequired values: novice→100, apprentice→200, journeyman→350, expert→500, master→750 (or keep current if already master)
  - If no userNodeMastery record exists for this node, create one with novice level
  - Return { score, earnedXP, newMastery: { currentLevel, xpCurrent, xpRequired }, leveledUp: boolean }
- If project:
  - Create challengeSubmission with status 'pending', response: { description, url }
  - Return { submitted: true, status: 'pending' }
